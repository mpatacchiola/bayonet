Detailed Description

Implementation of the Gibbs sampler.

The Gibbs sampler is part of the Markov Chain Monte Carlo (MCMC) methods. These family of methods work differently from rejection and LW sampling. Each sample is generated by making a random change to the preceding one. The algorithm starts with a random state where only the evidence variables are fixed at their observed values. After the initialization, the sampling is done conditioned on the current values of the variables in the Markov blanket of the node. The algorithm then continue moving at random, flipping one variable at a time but leaving the evidences unchanged.

This method is different from the same methods in the other samplers. The first sample is obtained at random. The next samples are choosen for each node picking up a value from the Markov blanket of the node. This probability is proportional to the probability of the variable given its parents times the probability of each child given its respective parents.
